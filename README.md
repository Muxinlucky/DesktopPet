# SmartTodo Assistant

[English](README_en.md)

> 基于大模型意图解析与系统音频感知的桌面智能助理 -- Tauri v2 + Rust 内核，霓虹悬浮 UI，零框架纯 DOM 渲染。

## 核心特性

### 智能意图代办

接入 DeepSeek 等大模型 API，自然语言输入自动分类为"闹钟提醒"或"永久备忘"。

```
"30 分钟后提醒我喝水"    -->  倒计时提醒，到时循环提示音 + 视觉脉冲
"学习深度学习"           -->  永久备忘，存入 localStorage
"1 小时后叫我开会"       -->  倒计时提醒
```

模糊时间表达回退到 LLM 语义解析，无需精确格式。

### AI 人格对话

连接任意 OpenAI 兼容接口，六种内置性格预设 + 自定义性格面板：

| 预设 | 风格 |
|------|------|
| 傲娇猫 | 外冷内热，偷偷关心 |
| 阳光少女 | 活力充沛，感叹号密度溢出 |
| 毒舌朋友 | 吐槽精准，但没有恶意 |
| 温柔前辈 | 暖心耐心，轻声细语 |
| 中二少年 | 戏剧独白，中二台词 |
| 禅系柴犬 | 佛系哲学，惜字如金 |

首次点击触发完整 LLM 回复，之后回落到轻量语录，10 分钟冷却后再次触发 -- 兼顾体验与成本。

### 系统音频感知

基于 Windows Core Audio API 的底层峰值检测，实时捕获系统音频输出电平。音乐播放时宠物随节拍飘出音符粒子，工作时保持安静。

### 极客美学 UI

暗黑霓虹风格悬浮面板 -- 毛玻璃背景、圆角发光边框、定制霓虹滚动条。所有面板均为浮层设计，不侵入桌面布局。

### 系统深度集成

- 开机自启动（Tauri autostart 插件）
- 窗口置顶（状态跨会话持久化）
- 文件拖入回收站（拖到宠物身上即可删除）
- GitHub 提交监控（有新 commit 时宠物冒出鼓励语）

### 专注模式

5 / 15 / 30 / 45 / 60 分钟番茄钟计时器。专注期间宠物保持安静、显示倒计时，结束后铃声提醒。

## 快速开始

**安装**

前往 [Releases](../../releases) 页面下载最新安装包（`.msi` 或 `setup.exe`），安装后启动即可。

**配置 API**

右键宠物 > 对话模式 > 接入 API，填入以下信息：

| 字段 | 说明 |
|------|------|
| Endpoint | API 服务地址（支持任何 OpenAI 兼容接口，自动补全 `/v1`） |
| Key | 你的 API 密钥 |
| Model | 模型名称（默认 `gpt-3.5-turbo`） |

点击"保存"自动测试连通性，绿色表示成功。

## 基本操作

| 操作 | 效果 |
|------|------|
| 左键点击 | 宠物打招呼，冒出粒子特效 |
| 按住拖拽 | 宠物跟着鼠标跑，松手落地弹跳 |
| 右键菜单 | 打开功能菜单 |
| 拖文件到宠物身上 | 文件送入回收站 |
| 长时间不操作 | 宠物发呆、思考、最后睡着 |
| 移动鼠标唤醒 | 宠物惊喜跳起 |

## 开发与构建

```bash
npm install            # 安装依赖
npm run tauri dev      # 开发模式（热重载）
npm run tauri build    # 生产构建（生成安装包）
```

构建产物位于 `src-tauri/target/release/bundle/nsis/`。

## 技术规范

- **运行时**: Tauri v2 (Rust 后端 + WebView2 前端)
- **前端**: Vite + TypeScript, 零框架纯 DOM 渲染
- **动画**: CSS `steps()` 精灵图引擎，无 Canvas/WebGL
- **安装包**: NSIS, 可自定义安装目录

详细架构文档见 [ARCHITECTURE.md](ARCHITECTURE.md)。

## 宠物包格式

支持导入自定义宠物（`.zip` 文件）：

```
pet.json              -- 宠物清单（id, displayName, description, spritesheetPath, version）
spritesheet.webp      -- 精灵图（8列 x N行，每格 192x208）
```

右键 > 设置 > 导入宠物(.zip) 即可更换。
